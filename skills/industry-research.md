# Industry Investment Research: Supply-Chain Panorama + Four-Master Single-Stock Framework

Conduct systematic supply-chain investment research on the $ARGUMENTS industry.

## Research Goals

Starting from an investment theme / logic chain, complete:
1. Validate every link of the investment logic chain
2. Draw a complete supply-chain panorama
3. Scan all listed companies globally (A-shares/HK/US/international)
4. Run the four-master framework on the leading companies in each sub-link
5. Output an industry-level portfolio allocation recommendation

---

## Step 1: Build and Validate the Investment Logic Chain

### 1.1 Draw the logic chain
Use an arrow chain to express the causality from "underlying trend" to "beneficiary target", e.g.:
```
Underlying trend A
    → drives demand B
        → creates bottleneck/hard demand C
            → beneficiary supply chain D
```

### 1.2 Validate link by link
Question each arrow of the logic chain and seek evidence:

| Link | Core assumption | How to validate | Data source |
|------|---------|---------|---------|
| A→B | | Search industry data/forecasts | |
| B→C | | Search supply-demand analysis | |
| C→D | | Search real cases/contracts | |

### 1.3 Find "validation events that have already happened"
List the **real business events that have been signed/landed** supporting the logic chain (not forecasts), e.g. big-company purchase agreements, policy documents, industry reports.

## Step 2: Draw the Supply-Chain Panorama

### 2.1 Draw the supply-chain structure
Decompose the industry into upstream → midstream → downstream → ancillary, e.g.:
```
Upstream: raw materials / resource extraction → material processing / purification
Midstream: core equipment manufacturing → system integration / engineering → new-tech R&D
Downstream: operation / service → end customer
Ancillary: testing / certification → maintenance service → financial tools (ETF/trust)
```

### 2.2 Identify each link's "business characteristics"
Label each link:

| Link | Business model | Gross-margin range | Competitive structure | Barrier type | Cyclicality |
|------|---------|-----------|---------|---------|--------|
| | sell resource/equipment/service/rent | | monopoly/oligopoly/full competition | resource/license/tech/scale | strong/medium/weak |

### 2.3 Mark the "chokepoint links"
Identify the links in the supply chain where supply is tightest, substitution is hardest, and margins are highest — these are often where the best investment targets sit.

## AI Research Bias Awareness: Special Traps in Industry Research

In industry research, AI data bias amplifies in unique ways:

**Industry-level bias**:
| Bias type | Manifestation | Countermeasure |
|---------|------|------|
| Mature-industry bias | Traditional industries (banking/energy/consumer) have abundant material, AI analysis looks "more certain" | Certainty comes from the business model, not the number of reports |
| New-industry underrating | New industries (AI apps/synthetic biology, etc.) have scarce material, AI analysis skews conservative | Judge industry value with "endgame thinking", not "current data" |
| Large-cap bias | Big companies have far more material than small ones, AI naturally leans toward recommending leaders | Small companies may have a better risk/reward; don't ignore them just because the analysis is short |
| Listing bias | Scanning only listed companies misses key unlisted players in the chain | Must search unlisted companies, flag "future IPO candidate" |
| English-language bias | AI handles English material better, may underrate Chinese/Asian players | Must search both Chinese and English sources |

**Anti-bias measures in the supply-chain scan**:
1. For each link, don't only list "companies AI easily finds" — proactively search "obscure but possibly high-quality targets"
2. For info-scarce small caps, don't lower the recommendation just because the analysis is short — judge by core questions (business essence, moat, management), not report length
3. In the final report, label each company's "information sufficiency" (A/B/C grade) so the reader knows how reliable the AI analysis is

## Step 3: Global Listed-Company Scan

Use the Task tool to launch background agents to comprehensively search all listed companies in the industry.

### Search list
- US (NYSE/NASDAQ/NYSE American) relevant companies
- A-shares (Shanghai/Shenzhen) relevant companies
- HK relevant companies
- Other international markets (Japan/Korea/Europe/Australia, etc.)
- Industry ETFs
- Key unlisted companies (possible future IPOs)

### For each company, collect
- Company name (Chinese and English)
- Ticker and exchange
- Market cap (approximate)
- One-line description (its position and role in the supply chain)
- Whether a pure-play target (pure nuclear vs diversified company with a nuclear business)
- Which link of the supply chain it belongs to

### Output format
Classify by supply-chain link, one table per link, including all scanned companies.
Then stratify by investment certainty:
- **Tier 1**: large cap, pure-play, industry leader
- **Tier 2**: mid cap, pure-play or high share, sub-segment leader
- **Tier 3**: small cap, development stage, high-risk/high-elasticity
- **Tier 4**: large diversified company with a relevant business

## Step 4: Four-Master Analysis of the Leaders in Each Link

For the **Tier 1 and Tier 2 companies** in each supply-chain link, run the following (Tier 3/4 companies need only a brief comment):

### 4.1 Business essence (Duan Yongping)
- Define in one sentence what this company does in the supply chain
- Revenue structure and growth
- Gross/net margin level and trend
- Cash-flow characteristics
- **Probe**: is this a good business? Why?

### 4.2 Moat (Buffett)
Score the five moat types (★1-5):

| Moat | Strength | Evidence |
|--------|------|------|
| Brand / pricing power | | |
| Switching cost | | |
| Network effect | | |
| Scale effect | | |
| Tech / license barrier | | |

**Probe**: will the moat still be there in 10 years?

### 4.3 Risk (Munger)
- How is this company most likely to fail?
- What's it worth in the worst case?
- Why won't smart people buy it?

### 4.4 Management (Duan Yongping + Buffett)
- Who is the CEO/founder? Key decision record
- Stake and interest alignment
- Brief grade (A/B/C)

### 4.5 Valuation snapshot
- Current PE/PS/EV/EBITDA
- Comparison with competitors in the same link
- Brief verdict: expensive / fair / cheap

### 4.6 Recommendation level
Flag with ★1-5:
- ★★★★★ = core-position candidate
- ★★★★☆ = satellite-position candidate
- ★★★☆☆ = watchlist
- ★★☆☆☆ = high-risk option
- ★☆☆☆☆ = not recommended
